Cambiamenti nella versione 1.1:

- Fase Beta terminata, il programma è rilasciato in versione 1.1
- Nuovo formato TraduSCI2 per i file che si sostituisce al formato precedente.
  A causa di una limitazione ereditata dal formato originale di Sierra, il vecchio
  formato rendeva inutilizzabili i file più grandi di 64Kbytes. Il nuovo formato
  permette invece di memorizzare file di qualunque grandezza.
