Changes in version 1.3:

INTERFACE
- GUI and menus simplified
- New multi-column string list
- Clone button and clone info removed from the main GUI
- Multi-string clones are now displayed as list of strings
- New font for strings: "Courier New"

- Problem when saving and using a Case sorting
- Various problems with special characters in Glossary
- Error duplicating a sequence clone when using a sorting other than ByIndex
- Graph nodes in wrong order (often node 1)
- Error in MAP format used with vocal SYNCs
- Removing MAP reference, if void, while uncloning
- Various bugs on pathnames
- Other glitches

OTHER
- Function to import/export text files containing user-defined labels
- Playing of character voices
- Automatic detection of temporary files left after a program crash
- Loading a SCI VOlume is now delegated to an external DLL module
- New, simpler manual
- Spanish version

________________________________________________________
Changes in version 1.1:

- Beta stage terminated, program is now released as version 1.1
- New file format TraduSCI2 is introduced in place of the previous one.
  The old format was corrupting files bigger than 64Kbytes due to a limitation
  inherited from the original Sierra file format. The new format allows you to
  store files of any size you need, instead.
- A few minor improvements in the user interface
