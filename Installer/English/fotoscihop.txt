FotoSCIhop 1.3 - (c) Enrico Rolfi, 2004-2021

This program will let you able to modify images from SCI1.1 and SCI32 games.

Through an easy-to-use user interface, you will be able to replace the original images with
modified copies. New images should have the same color table as the original and must
be in 256 colors. It's important to not change the original colors so that there will be no
drawing problems in the game. Keep in mind that certain colors are more important than
others, and some of them are transparent or translucent too.

It is not possible to make brand new picture files using this program, but you can only
view and modify existing ones.
